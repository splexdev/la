Guia de Comandos

Aprenda a utilizar os comandos de consulta do @rockconsultasgratisbot.

Comandos de Consulta

/cpf <número>

Consulta informações detalhadas de um CPF.
Exemplo: /cpf 12345678900

Usar no Bot

/cnpj <número>

Consulta dados cadastrais de uma empresa (CNPJ).
Exemplo: /cnpj 12345678000100

Usar no Bot

/nome <texto>

Busca informações vinculadas a um nome completo.
Exemplo: /nome Joao da Silva

Usar no Bot

/telefone <número>

Procura dados associados a um número de telefone (com DDD).
Exemplo: /telefone 11987654321

Usar no Bot

/email <endereço>

Consulta informações a partir de um e-mail.
Exemplo: /email email@dominio.com

Usar no Bot

/placa <placa>

Verifica dados de um veículo (padrão Mercosul ou antigo).
Exemplo: /placa ABC1234

Usar no Bot

/cnh <cpf>

Consulta dados da CNH a partir do CPF do titular.
Exemplo: /cnh 12345678900

Usar no Bot

/foto <cpf>

Busca a foto de um CPF (disponível em bases específicas).
Exemplo: /foto 12345678900

Usar no Bot

/ip <endereço>

Obtém informações de um endereço IP.
Exemplo: /ip 8.8.8.8

Usar no Bot

/cep <número>

Consulta endereço e, se disponível, moradores de um CEP.
Exemplo: /cep 01001000

Usar no Bot

Comandos de Conta e Planos

/start

Inicia o bot e exibe o menu principal com seu status e plano atual.

Usar no Bot

/resgatar <key>

Ativa um plano de acesso usando uma chave de ativação.
Exemplo: /resgatar MINHACHAVE12345

Usar no Bot
